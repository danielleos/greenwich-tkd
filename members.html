Do your best to remember the Korean terminology, the meanings of the belt colours and patterns, as well as performing your patterns as well as you possibly can (this means ALL the patterns up until your current grade - not just your current one).

White Belt

Grading Syllabus: Promotion from 10th to 9th kup
  1. Saju Jirugi / Four Directional Punch [Watch Video]
  2. Saju Makgi / Four Directional Block [Watch Video]
  3. Narani so kaunde jirugi
    Parallel stance middle punch
  4. Moving Forwards & Backwards
    • Gunnun so kaunde jirugi
      Walking stance middle punch
    • Gunnun so najunde yop bakat palmok makgi
      Walking stance low side outer forearm block
    • Gunnun so najunde yop sonkal makgi
      Walking stance low side knife-hand block
    • Gunnun so kaunde yop an palmok makgi
      Walking stance middle side inner forearm block
  5. Spot exercises in Walking stance (Gunnun sogi)
    • Ap cha olligi / Front rising kick
    • Golcho olligi / Hooking rising kick
    • Ap cha busigi / Front snap kick
    • Dollyo chagi / Turning kick
  6. Sambo Matsogi / Three-Step Sparring
    Demonstrate the following 2 elements of Three-Step Sparring
    • Walking stance middle obverse punch
    • Walking stance inner forearm middle block
  7. Flexibility Test - Show stretching ability
  8. Power Test - Push ups Number set by examiner
  9. Theory - Knowledge of Tae Kwon-Do [Theory Cheat Sheet]

Yellow Tag

Grading Syllabus: Promotion from 9th to 8th kup
  1. All elements up to 9th kup
  2. Chonji Tul [Watch Video]
  3. Acquaintance of Dan-Gun
  4. Moving Forwards & Backwards
    • Niunja so palmok daebi makgi / L-stance forearm guarding block
    • Niunja so sonkal daebi makgi / L-stance knife-hand guarding block
    • Niunja so sonkal yop taerigi / L-stance knife-hand side strike
    • Niunja so sang palmok makgi / L-stance twin forearm block
  5. Spot exercise in L-stance
    • Naeryo chagi / Downward kick
    • Yop cha jirugi (tora) / Side piercing kick (turn)
      (After pivoting 180 degrees on the ball of the front foot)
  6. Spot exercise in L / Parallel stance
    • Dwit cha jirugi / Back piercing kick
  7. Side / Forward Stepping Sitting / L-stance
    • Yop cha jirugi / Side piercing kick
  8. In an L-stance
    • Twimyo dollyo chagi / Flying turning kick
  9. L-stance / Parallel stance
    • Twimyo yop chagi / Flying side kick
  10. Sambo Matsogi / Three-Step Sparring
    With a partner, demonstrate 6 elements in walking stance and L-stance
    • Walking stance middle obverse punch
    • Walking stance Inner forearm middle block
    • Walking stance varied counter attack
    • L-stance middle reverse punch
    • L-stance Inner forarm middle block
    • L-stance varied counter attack
  11. Flexibility Test - Show stretching ability
  12. Power Test - Push ups - Number set by examiner
  13. Theory - Knowledge of Tae Kwon-Do [Theory Cheat Sheet]

Blue Tag

      L-stance backfist high side strike
    • Niunja so bandae kaunde jirugi
      L-stance reverse middle punch
    • Gunnun so sonbadak noollo makgi
      Walking stance palm pressing block
    • Gunnun so sang joomuk sewo jirugi
      Walking stance twin fist vertical punch
    • Gunnun so dwijibo jirugi
      Walking stance upset punch
    • Gunnun so kyocha joomuk chookyo makgi
      Walking stance X-fist rising block
    • Gunnun so wi palkup taerigi
      Walking stance upper elbow strike
    • Dwitbal so sonbadak ollyo makgi
      Rear foot stance palm upward block
    • Gojang so degutja makgi
      Fixed stance U-shape block
    • Twimyo dung joomuk yop taerigi
      Flying backfist side strike
    • Twimyo bandae dollyo chagi
      Flying reverse turning kick
  5. Special Techniques
    • Twimyo nopi ap cha busigi / Flying overhead front snap kick
      (1 board height of raised arm - good balance)
    • Twimyo nomo yop chagi / Flying long side kick
      (1 board - male over 3 persons, female over 2 persons - good balance)
  6. Power Test
    • Gunnun so ap joomuk jirugi / Walking stance forefist punch
      (1 board)
    • Niunja so sonkal yop taerigi / L-stance knife-hand side strike
      (1 board)
    • Ap cha busigi / Front snap kick
      (2 boards)
    • Yop cha jirugi / Side piercing kick
      (2 boards)
    • Push ups - Number set by examiner
  7. The examiner will choose 4 destructions (2 from special techniques and 2 from power test including 1 hand technique)
  8. Ilbo Matsogi - One-Step Sparring (10 elements)
  9. Ban-Jayoo Matsogi / Semi-Free Sparring
  10. Jayoo Matsogi / Free Sparring

Won-Hyo

[Click for English]
Junbi jase: Moa junbi sogi A
  1. Orun niunja so sang palmok makgi
  2. Orun niunja so nopunde sonkal anuro taerigi (gwa wen joomuk e orun okkae)
  3. Wen gojung so kaunde yop jirugi
  4. Wen niunja so sang palmok makgi
  5. Wen niunja so nopunde sonkal anuro taerigi (gwa orun joomuk e wen okkae)
  6. Orun gojung so kaunde yop jirugi
