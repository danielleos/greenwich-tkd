White Belt

Grading Syllabus: Promotion from 10th to 9th kup
  1. Saju Jirugi / Four Directional Punch [Watch Video]
  2. Saju Makgi / Four Directional Block [Watch Video]
  3. Narani so Kaunde Jirugi / Parallel Stance Middle Punch
  4. Moving Forwards & Backwards
    • Gunnun so kaunde jirugi / Walking stance middle punch
    • Gunnunn so bakat palmok najunde yop makgi / Walking stance outer forearm low side block
    • Gunnun so sonkal najunde yop makgi / Walking stance knifehand low side block
    • Gunnun so an palmok kaunde yop makgi / Walking stance inner forearm middle side block
  5. Spot exercises in Walking stance (Gunnun sogi)
    • Apcha olligi / Front rising kick
    • Golcho olligi / Hooking rising kick
    • Apcha busigi / Front snap kick
    • Dollyo chagi / Turning kick
  6. Sambo Matsogi / Three-Step Sparring
    Demonstrate the following 2 elements of Three-Step Sparring
    • Walking stance middle obverse punch
    • Walking stance inner forearm middle block
  7. Flexibility Test - Show stretching ability
  8. Power Test - Push ups Number set by examiner
  9. Theory - Knowledge of Tae Kwon-Do [Learn Theory]

Yellow Tag

Grading Syllabus: Promotion from 9th to 8th kup
  1. All elements up to 9th kup
  2. Chonji Tul [Watch Video]
  3. Acquaintance of Dan-Gun
  4. Moving Forwards & Backwards
    • Niunja so palmok daebi makgi / L-stance forearm guarding block
    • Niunja so sonkal daebi makgi / L-stance knifehand guarding block
    • Niunja so sonkal yop taerigi / L-stance knifehand side strike
    • Niunja so sang palmok makgi / L-stance twin forearm guarding block
  5. Spot exercise in L-stance
    • Naeryo chagi / Downward kick
    • Yopcha jirugi (tora) / Side piercing kick (turn) (After pivoting 180 degrees on the ball of the front foot)
  6. Spot exercise in L / Parallel stance
    • Dwitcha jirugi / Back piercing kick
  7. Side / Forward Stepping Sitting / L-stance
    • Yopcha jirugi / Side piercing kick
  8. In an L-stance
    • Twimyo dollyo chagi / Flying turning kick
  9. L-stance / Parallel stance
    • Twimyo yop chagi / Flying side kick
  10. Sambo Matsogi / Three-Step Sparring
    With a partner, demonstrate 6 elements in walking stance and L-stance
    • Walking stance middle obverse punch
    • Walking stance Inner forearm middle block
    • Walking stance varied counter attack
    • L-stance middle reverse punch
    • L-stance Inner forarm middle block
    • L-stance varied counter attack
  11. Flexibility Test - Show stretching ability
  12. Power Test - Push ups - Number set by examiner
  13. Theory - Knowledge of Tae Kwon-Do [Learn Theory]

Blue Tag

  3. Acquaintance of Joong-Gun
  4. Moving Forwards & Backwards
    • Niunja so sonkal dung kaunde bakuro makgi / L-stance reverse knifehand middle outward block
    • Niunja so dung joomuk nopunde yop taerigi / L-stance back fist high side strike
    • Niunja so bandae kaunde jirugi / L-stance reverse middle punch
    • Gunnun so sonbadak noollo makgi / Walking stance palm pressing block
    • Gunnun so sang joomuk sewo jirugi / Walking stance twin fist vertical punch
    • Gunnun so dwijibo jirugi / Walking stance upset punch
    • Gunnun so kyocha joomuk chookyo makgi / Walking stance X-fist rising block
    • Gunnun so wi palkup taerigi / Walking stance upper elbow strike
    • Dwitbal so sonbadak ollyo makgi / Rear foot stance palm upward block
    • Gojang so degutja makgi / Fixed stance U-Shape block
    • Twimyo dung joomuk yop taerigi / Flying back fist side strike
    • Twimyo bandae dollyo chagi / Flying reverse turning kick
  5. Special Techniques
    • Twimyo nopi apcha busigi / Flying overhead front snap kick (1 board height of raised arm - good balance)
    • Twimyo nomo yop chagi / Flying long side kick (1 board - male over 3 persons, female over 2 persons - good balance)

    • Yopcha jirugi / Side piercing kick (2 boards)

    • Niunja so doo palmok najunde miro makgi
      L-stance double forearm low pushing block

      X-stance X-fist pressing block

      Sitting stance outer forearm W-shape block
    • Moa so dung joomuk yopdwi taerigi
      Closed stance back fist side back strike (Spot Exercise)
